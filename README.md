Description
------------
Anonymous imageboard, continued development of the infinity engine.


Requirements
------------

* Linux like OS
* PHP >= 5.6 
* MYSQL / MariaDB 
* ffmpeg, memcached (optional: graphicsmagick, gifsicle, imagemagick,  exiftool)
